## 这是青海大学暑期学期《程序设计案例实训》大作业。

# PVZ2677

> 基于 C++ 和 EasyX 开发的《植物大战僵尸 Lite》课程项目。

## 项目简介

本项目目标是在学习软件工程开发流程的基础上，实现一个简化版《植物大战僵尸》。

---

## 开发环境

- Visual Studio 2022
- C++
- EasyX
- Git

---

## 项目目录

```text
assets/     游戏资源
docs/       项目文档
include/    头文件
project/    VS2022 工程
src/        源代码
```
